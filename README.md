Project Summary
This project focused on developing a C++ program that analyzes grocery item frequency data from an input file. The program reads a list of purchased items, counts how often each item appears, and outputs both a frequency report and a histogram-style visualization. The goal was to simulate a practical scenario such as retail inventory tracking or sales trend analysis. Completing this project demonstrates my ability to work with file input/output, utilize STL data structures like maps, and design a modular, multi-file C++ application.

What I Did Well

By separating logic into header and implementation files, I kept main.cpp focused on high-level program flow while delegating data processing and file handling to dedicated classes and functions.
